Projet - Bon de commande

Sujet : Une application web de gestion de stock des voitures.

On a une base de données MySQL.
On fait la gestion des voitures, commandes et factures.

Fonctionnalités :

- afficher toutes les commandes;
- afficher les détails des commandes (nom du produit, prix unitaire, quantité, prix total);
- afficher les détails du client qui a effectué la commande.
